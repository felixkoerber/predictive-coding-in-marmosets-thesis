# Code for Master's Thesis "Spectral Transfer Entropy Reveals Predictive-Coding Network Dynamics of Auditory Deviance Detection in Common Marmosets"
### Felix Körber
#### Supervision: Alessandro Tavano, Cosimo Iaia

For Analysis of Information Transfer in ECoG Marmoset data following an Auditory Roving Oddball Task available at https://neurotycho.org/download-marmoset-ecog

Script 2 and 3 require IDTxl (Step 3 https://github.com/pwollstadt/IDTxl/tree/test_spectral_te).
Step 2 is designed to be run on a computing cluster with CPUs and can be called by executing the .sh file. Requires ACME
